# Bloxd ドット絵コンバーター / Bloxd Pixel Art Converter
画像からBloxd用のドット絵を作成するコンバーター
## 概要
このコンバーターは、自分でアップロードした画像から、Bloxd用のCodeとして変換し、Bloxdでドット絵を作るコンバーターです

----
## コード本体
コード本体は`index.html` `style.css` `script.js` に掲載してあります
- [index.html](/index.html)
- [style.css](/style.css)
- [script.js](/script.js)
